This project aims to develop a predictive model to estimate prices based on various input features using machine learning techniques. We utilize a comprehensive web scraped dataset comprising multiple features, which serve as predictors for our target variable, the price. The primary objective is to build an accurate and efficient model that can provide reliable price predictions, aiding in better decision-making processes.
